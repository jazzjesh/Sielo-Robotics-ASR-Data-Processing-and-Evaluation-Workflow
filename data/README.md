# Data

This folder contains documentation and sample metadata for the datasets used in this project.

## Dataset Stages
- **Stage 1:** One-word command data
- **Stage 2:** Clean full-sentence dysarthric-style speech
- **Stage 3:** Noisy full-sentence dysarthric-style speech

## Notes
The complete dataset is not uploaded to this repository due to file size and research/project constraints.

This folder may include:
- sample metadata files
- example dataset structure
- documentation describing splits and usage
